# FanchmWrt x86_64 Files Directory

This directory contains custom files that will be copied to the firmware during build.

## Structure

- `etc/` - System configuration files
  - `config/` - UCI configuration files
  - `openclash/` - OpenClash configuration
  - `uci-defaults/` - Default settings scripts

## Usage

Any files placed in this directory will be copied to the firmware root during the build process.
